System Architecture

Tiered Modular Model

A decoupled four-layer framework ensuring high horizontal scalability and fault tolerance.

  • Perception: ESP32 + Heterogeneous Sensor Array
  • Logic: Python Flask REST Gateway
  • Analytics: Real-time Chart.js Telemetry
  • Response: SG90 Servo Actuation

Hardware Specification
