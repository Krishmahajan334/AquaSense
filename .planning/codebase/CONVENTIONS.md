# Coding Conventions

## JavaScript
- **Style:** Vanilla ES6+, no external frameworks (React/Vue).
- **DOM Access:** Direct `document.getElementById` and `querySelector`.
- **Async:** `fetch()` with `async/await` for API calls.
- **Organization:** Page-specific JS files (e.g., `admin.js`, `analytics.js`).

## Python
- **Framework:** Flask with route decorators.
- **Variable Naming:** `snake_case` for variables and functions.
- **Comments:** Focused on module intent and complex logic.

## CSS
- **Design System:** Custom CSS variables for colors (Dark theme: `#0a0a0a`, `#1a1a1a`).
- **Layout:** Flexbox and Grid based responsive design.
- **Glassmorphism:** Use of `backdrop-filter: blur()` and semi-transparent backgrounds.

## Git
- **Commits:** GSD-style atomic commits for task tracking.
