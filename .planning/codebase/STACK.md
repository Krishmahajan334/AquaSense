# Technology Stack

## Backend
- **Language:** Python 3.x
- **Framework:** Flask 3.0.0
- **Server:** Gunicorn 21.2.0
- **Database:** Google Cloud Firestore (Primary), Local CSV (`history.csv`) for simulation/history.

## Frontend
- **Languages:** HTML5, CSS3, JavaScript (Vanilla ES6+)
- **Architecture:** Client-side rendering with AJAX/Fetch to Python backend.
- **UI Style:** Custom CSS with Glassmorphism/Premium dark mode elements.

## Hardware Simulation
- **Language:** C++
- **Simulator:** `simulator.cpp` (Custom simulator for water quality/consumption telemetry).

## DevOps & Tools
- **Containerization:** Docker (`Dockerfile`)
- **Intelligence:** GSD (Get Shit Done), Graphify
- **Communication:** Telegram Bot API
