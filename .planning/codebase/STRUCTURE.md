# Directory Structure

```text
.
├── simple_mockup/            # Main application directory
│   ├── backend/              # Python Flask API
│   │   ├── server.py         # Entry point
│   │   ├── requirements.txt  # Dependencies
│   │   └── config.json       # Backend settings
│   ├── frontend/             # Web assets (HTML/CSS/JS)
│   │   ├── index.html        # Main dashboard
│   │   ├── admin.html        # Admin panel
│   │   └── styles.css        # Visual styling
│   └── hardware_sim/         # C++ Telemetry Simulation
│       └── simulator.cpp     # Sensor simulation logic
├── Diagrams/                 # UML and System Diagrams
├── graphify-out/             # Knowledge Graph data
├── Aquasense-X/              # Hardware/CAD/Schematic files
└── .planning/                # GSD Planning Context
```
